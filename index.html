About me

I'm a Python developer who's decent at Javascript. My primary areas of expertise are Flask web apps and web scraping, but I've also got some experience with a lot of other things including, but not limited to, HTML, S/CSS, PostgreSQL, MongoDB, Sqlalchemy, Django, Scrapy, vue.js, webpack, the node/npm ecosystem, GNU/Linux and many widely used Python libraries.

I think good design is transparent, if not invisible, and preferably good looking, complex is better than complicated and a thing is done when nothing more can be taken away (without messing everything up, of course).

Since around 2011, I've been writing Python and Javascript and I've been using GNU/Linux for a while longer. I've been a freelance Spanish to English translator since 2007. I have a BFA from The Art Institute of Chicago and an MFA from the University of Pennsylvania.
